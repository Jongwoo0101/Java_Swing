# Java_Swing
```
import javax.swing.*;
```
이 프로젝트는 자바에서 GUI를 구현하기 위해 JDK에서 기본적으로 제공하는 개발 툴킷인 스윙을 이용합니다

# JDK 버전
OpenJDK 17 이상의 JDK환경에서 실행하는 것을 권장합니다.
